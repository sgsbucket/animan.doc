.. _createResource:

Create Resource
====================

Privilege required:
 - admin
 - or EDIT RESOURCE privilege

Location:
 - (page) Show / (tab) Resources
 - (page) Episode / (tab) Resources

Resource helps studio to manage a resource file that may be an image, a video, an animation, or a storyboard.

To create a resource:

#. Click "+" button in the resources page
#. Choose "asset" or "funpack"
#. Enter the name of the resource
#. Choose a Category (this determines what file type this resource is)
#. Optionally enter a sub-category
#. Optionally enter tags for easier search
#. Click "Create" button
