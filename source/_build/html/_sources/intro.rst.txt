Introduction
============
Animan is a web app that helps studios manage the production of broadcast animation.

Animan works with its companion software Renderman to control and measure production from storyboard to 4k render.

Features
+++++++++++
#. Online and Offline Rendering
#. Work Statistics and Quota System
#. Workflow Management
#. Team Management, Messaging, and Logging
#. Revisions
#. Workfile Version Control and Backup
